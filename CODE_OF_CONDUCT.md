# Contributor Covenant Code of Conduct

Our community is dedicated to providing an open and welcoming experience for everyone. We expect participants to:

- Be respectful and kind
- Assume good faith
- Focus on the work, not the person

If you experience or witness unacceptable behavior, contact the maintainers at: mikejohanning@gmail.com

This project follows the Contributor Covenant; more details: https://www.contributor-covenant.org
